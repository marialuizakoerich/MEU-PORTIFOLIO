# Portfólio — Maria Luiza Koerich

Portfólio pessoal desenvolvido em **HTML, CSS, JavaScript, GSAP**, com foco em **UI** e **animações com GSAP**.  
O projeto possui navegação por âncoras com scroll suave (ScrollSmoother) e efeitos visuais para destacar seções como Jornada, Cursos, Tecnologias, Projetos e Contato.

---

## ✨ Preview / Demo

🌐 **Acesse online pela GitHub Pages:**  
👉 [https://marialuizakoerich.github.io/MEU-PORTIFOLIO/](https://marialuizakoerich.github.io/MEU-PORTIFOLIO/)



## 🧩 Funcionalidades

- **Header com navegação por âncoras** (Minha Jornada, Cursos, Tecnologias, Projetos e Contato)
- **Scroll suave** com `ScrollSmoother`
- **Scroll com efeitos** usando `ScrollTrigger`
- **Animação flutuante** na imagem da hero (`.tec_img`)
- Seções organizadas em layout moderno:
  - Minha Jornada (cards em timeline)
  - Cursos (grid responsivo)
  - Tecnologias (grid + soft skills)
  - Projetos (cards com descrição, tags e link)
  - Contato (perfil + formulário)

---

## 🛠️ Tecnologias usadas

- **HTML5**

- **GSAP (GreenSock)**
  - `ScrollTrigger`
  - `ScrollSmoother`

## ▶️ Como rodar o projeto

### Opção 1 — Abrir direto no navegador
1. Baixe/cloque o repositório
2. Abra o arquivo `index.html` no navegador


### Opção 2 — Rodar com Live Server (recomendado)
Se você usa VS Code:
1. Instale a extensão **Live Server**
2. Clique com o botão direito no `index.html`
3. Selecione **Open with Live Server**
